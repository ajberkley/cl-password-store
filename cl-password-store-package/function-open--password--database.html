Function Open-Password-Database

Part of:

package cl-password-store
( open-password-database &key (spec '((cl-password-store.sqlite3) database-type sqlite3 if-exists old)) )

Open password database specified by :SPEC, which must be and argument list suitable for clsql:connect. Default is to use '(("cl-password-store.sqlite3") :sqlite3); if you use this default you will have to have clsql-sqlite3 loaded too. Returns an clsql:database object suitable as argument to OPEN-PASSWORD-STORE.
